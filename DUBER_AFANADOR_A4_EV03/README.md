# Frontend Marketplace - Evidencia AA4_EV03
Proyecto frontend en React (Vite) para la evidencia GA7-220501096-AA4-EV03.

## Requisitos
- Node.js (14+)
- npm

## Ejecutar localmente
1. `npm install`
2. `npm run dev`
3. Abrir `http://localhost:5173/` (Vite mostrará la URL si difiere)

## Estructura
- src/components: componentes reutilizables
- src/data: datos mock para pruebas
- src/styles.css: estilos simples
- package.json: scripts y dependencias

## Entrega
- Subir repo a GitHub y añadir `repo-link.txt`
